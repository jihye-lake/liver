
 : 지금까지 컴포넌트를 생성하고 컨테이너에 등록해 출력하는 방법을 알아 보았습니다.
   그런데 문제가 있습니다.
   컨테이너에 컴포넌트를 출력할 때 어떻게 배치할 것인지를 지정하지 않았습니다.
   자바는 일정한 모양으로 컴포넌트를 재배치 할 수 있도록 하기위해 Layout Manager 를 제공합니다.
   레이아웃은 컴포넌트를 어떻게 배치할 것인가를 정하는 규격이고, 이를 설정하는 것이 레이아웃 매니저입니다.
   
   자바에서 컴포넌트 위치를 결정하는 방법은 두가지가 있습니다.
   
   1. 컨테이너에 add() 메소드를 사용하는 순서에 따라 위치가 결정됩니다.
   2. 레이아웃 매니저에 따라 위치가 정해집니다.
   
   자바는 Container 는 컴포넌트의 일종이기 때문에 컨테이너에 또다른 컨테이너를 등록할 수 있습니다.
   컨테이너는 프레임, 패널, 윈도우, 에플릿 등이 있습니다.
   결국 레이아웃 매니저는 컨테이너를 포함한 모든 컴포넌트의 배치에 관여합니다.
   자바에서 제공하는 레이아웃 관리자는
   FlowLayout, GridLayout, BorderLayout, CardLayout 등이 있습니다.
   
   레이아웃 매니저는 컴포넌트를 어떻게 배치하는 지를 결정하는 클래스입니다.
   그렇기 때문에 컨테이너를 제외한 모든 컴포넌트의 위치에 관여합니다.
   컨테이너에 또다른 컨테이너가 등록되어 있다면 서브 컨테이너도 레이아웃 매니저에 의해 위치가 결정됩니다.
