
String

 : 자바에서 문자열을 지정하는데 사용되는 객체는 String, StringBuilder, StringBuffer 세 가지가 있습니다.
   문자는 char cc = 'A' 처럼 작은 따옴표를 사용하지만 문자열은 String str = "seoul" 처럼 큰 따옴표를 처리합니다.
   프로그램에서 문자열을 많이 사용해 String 클래스는 마치 기본 자료형처럼 사용됩니다.
   String 클래스는 한 번 생성되면 변하지 않는 문자열 즉, 상수 문자열을 처리하는 클래스입니다.
   계속 변하는 문자열은 StringBuffer 클래스를 이용해 처리하는 것이 좋습니다.
   자바는 String 클래스를 사용할 때 내부적으로 StringBuffer 클래스로 변형되어 처리합니다.
   
   문자열 저장 객체 비교
